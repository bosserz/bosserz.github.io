SAT Math Full Pack

50 ชั่วโมง - ครบทุก Topic

สำหรับผู้ที่ต้องการสร้างพื้นฐานแข็งแรง + เทคนิคครบถ้วน + ฝึกข้อสอบจริง เหมาะกับผู้เริ่มต้นหรือเป้าหมาย 700+

  • ครอบคลุม Algebra, Geometry, Advanced Math, Data Analysis
  • เทคนิค Desmos ทำข้อยากให้ไว
  • Mock test + error analysis
  • เหมาะกับ: เริ่มต้นตั้งแต่ศูนย์ หรือมีพื้นฐานแต่ต้องการ 700+

SAT Math Foundation

30 ชั่วโมง - สร้างพื้นฐาน

เน้นสร้างพื้นฐาน Concept ให้แข็งแรง + เทคนิคเบื้องต้น เหมาะกับผู้ที่มีเวลาน้อยหรือเป้าหมาย 600-650

  • หัวข้อหลัก: Algebra + Geometry + Data
  • เทคนิคพื้นฐาน + ข้อง่ายถึงปานกลาง
  • Mini mock + review สำคัญ
  • เหมาะกับ: เป้าหมาย 600-650 หรือเวลาจำกัด

SAT Math Turbo

20 ชั่วโมง - เร่งด่วนทันสอบ

คอร์สเข้มข้นสำหรับผู้ที่มีวันสอบใกล้ เน้น High-ROI topics + Trick + Desmos ที่ช่วยเพิ่มคะแนนเร็ว

  • เน้น High-ROI topics + Pattern ที่ออกบ่อย
  • Desmos เทคนิคลัด ทำให้ไว
  • Speed strategy + timing
  • เหมาะกับ: มีพื้นฐานดี สอบใกล้ ต้องการ boost
